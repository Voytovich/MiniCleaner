# MiniCleaner

Приложение для очистки Windows с графическим интерфейсом. Помогает найти и удалить большие/старые файлы, а также исправить поврежденные записи в реестре Windows.

**Языки:** [English](README.md) | [Русский](README.ru.md) | [Українська](README.uk.md)

## Возможности

### Сканирование и удаление файлов
- Поиск больших файлов по заданному размеру
- Фильтрация по возрасту файлов
- Поиск "мусорных" файлов (логи, временные файлы, кеши и т.д.)
- Два режима отображения: по файлам или по папкам
- Безопасное удаление с подтверждением

### Исправление реестра Windows
- Поиск поврежденных записей в автозагрузке (Run/RunOnce)
- Поиск несуществующих путей в записях Uninstall
- Удаление проблемных записей из реестра

## Требования

- **ОС**: Windows 10/11

## Установка

1. Скачайте `MiniCleaner.exe` из раздела [Releases](https://github.com/Voytovich/MiniCleaner/releases)
2. Запустите `MiniCleaner.exe` — установка не требуется

## Использование

### Вкладка "Файлы"

1. **Выберите папку для сканирования**
   - Введите путь вручную (например, `C:\`) или нажмите "Обзор..."
   - Можно указать весь диск или конкретную папку

2. **Настройте фильтры**
   - **Мин. размер**: минимальный размер файлов для поиска (в МБ)
   - **Старше**: искать только файлы старше указанного количества дней
   - **Только подозрительный мусор**: искать только файлы с расширениями (.log, .tmp, .bak и т.д.)
   - **Пропускать системные папки**: автоматически исключает C:\Windows, C:\Program Files и т.д.

3. **Выберите режим отображения**
   - **Файлы**: список всех найденных файлов
   - **Папки**: группировка по папкам с общим размером

4. **Запустите сканирование**
   - Нажмите кнопку "Сканировать"
   - Прогресс отображается в строке состояния
   - Можно отменить сканирование кнопкой "Отменить"

5. **Удаление файлов**
   - Отметьте нужные файлы/папки галочками в колонке "Удалить"
   - Нажмите "Удалить отмеченное"
   - Подтвердите удаление (это действие нельзя отменить!)

### Вкладка "Реестр"

1. **Выберите типы проверки**
- **Автозагрузка (Run/RunOnce)**: поиск поврежденных путей в автозагрузке
- **Uninstall (поврежденные записи)**: поиск несуществующих путей в записях удаления программ

2. **Запустите сканирование**
   - Нажмите "Сканировать реестр"
   - Дождитесь завершения сканирования

3. **Исправление записей**
   - Отметьте проблемные записи галочками
   - Нажмите "Исправить отмеченное"
- **Внимание**: изменение реестра может повлиять на систему. Рекомендуется создать точку восстановления перед исправлением.

## Поддерживаемые типы "мусорных" файлов

Приложение автоматически определяет следующие типы файлов как потенциальный мусор:

- `.log`, `.tmp`, `.temp`, `.bak`, `.old`, `.cache`
- `.ds_store`, `.crdownload`, `.part`, `.dmp`, `.err`
- `.chk`, `.swp`, `.gid`, `.~lock`, `.thumbs`
- `.ini`, `.etl`, `.sqm`, `.msi`, `.msp`

## Цветовая индикация

В таблице результатов файлы отображаются цветом в зависимости от размера:
- **Красный**: ≥ 1 GB
- **Оранжевый**: ≥ 100 MB
- **Желтый**: ≥ 10 MB

## Безопасность

- **Удаление файлов необратимо** — убедитесь, что выбранные файлы действительно не нужны
- **Изменение реестра** может повлиять на работу системы — создайте точку восстановления перед исправлением
- Приложение не требует прав администратора для сканирования, но некоторые операции с реестром могут требовать повышенных прав

## Исключаемые папки (при включенной опции)

При включенной опции "Пропускать системные папки" автоматически исключаются:
- `C:\Windows`
- `C:\Program Files`
- `C:\Program Files (x86)`
- `C:\ProgramData`
- `C:\$Recycle.Bin`
- `C:\Users\All Users`

## Решение проблем

### Предупреждение Windows SmartScreen

При первом запуске `MiniCleaner.exe` Windows SmartScreen может показать предупреждение: "Система Windows защитила ваш компьютер" с "Неизвестный издатель". Это нормально для неподписанных приложений.

**Чтобы запустить приложение:**
1. Нажмите **"Подробнее"**
2. Нажмите **"Выполнить в любом случае"**

**Почему это происходит:**
- Windows SmartScreen блокирует неподписанные приложения по умолчанию
- Это функция безопасности, но приложение безопасно для использования

### Приложение не запускается
- Убедитесь, что вы используете Windows 10/11
- Проверьте, не блокирует ли антивирус приложение
- Попробуйте запустить от имени администратора

### Сканирование работает медленно
- Это нормально для больших дисков
- Используйте фильтры (минимальный размер, возраст) для ускорения
- Исключите системные папки

### Нет доступа к некоторым файлам/папкам
- Некоторые системные файлы защищены Windows
- Запустите приложение от имени администратора для доступа к защищенным файлам
